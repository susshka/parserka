# JanitorAI Python Parser
Запуск через [GitHub Codespaces](https://github.com/features/codespaces).
## Быстрый старт
1. Нажмите кнопку **Code** → **Codespaces** → **Create codespace on main**
2. В терминале выполните:
`python parser.py "https://janitorai.com/scripts/adb43eef-53a0-4e26-a0fd-8d20a8544fbc"`
3. Дождитесь результата — ваш парсер запустится в облаке, и вы увидите вывод в терминале.
