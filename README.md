# parserka
Эта дичь сделана с помощью вайбкодинга! Если оно кривое - вините иишку, а еще лучше - исправьте сами.

1. Скачиваем c git, распаковываем

2. Открываем Chrome → chrome://extensions/

3. Включием "Режим разработчика"

4. Загружаем "распакованное расширение" — выбераем папку "parser"

5. После появления расширения в "мои расширения" тыкаем "Сведения"

6. Там тыкаем галку "Закрепить на панели инструментов"

7. Идем на дженитор, ищем лорбук. Заходим на страницу лорбука, проверяем, что в адресной строке браузера ссылка формата: 
`https://janitorai.com/scripts/много_цифор_и_букв`

8. Тыкаем по кнопке в расширении и ждем, автоматически начнется скачка
